Require :
---------
- git
- u3jit-core (Does not need after V1.0)
- sudo (root-commend)

Installation : 
--------------
```
git clone https://github.com/claxpoint/netbaan.git
```
```
cd netbaan
```

Run As Superuser

```
sudo ./netbaan.sh
```

Uninstall:
----------
```
sudo rm netbaan
or delete it on your directory-gui
```
