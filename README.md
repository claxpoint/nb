SWDNS is Tool for Easy Switch or Change DNS/Nameserver via resolv.conf

![](https://i.imgur.com/y0mct7g.gif)

Require :
---------
- make

Installation : 
--------------
```
git clone https://github.com/yanuarakhid/swdns.git
```
```
cd swdns && sudo make install
```

Run As Superuser

```
sudo swdns
```

Uninstall:
----------
```
sudo make uninstall
```
